## v-bind
* the full syntax

```  
    <img v-bind:src="dataProperty" />
```  
* the shorthand syntax

```  
    <img :src="dataProperty" />
```

## v-on
* the full syntax

```
    <button v-on:click="methodName"></button>
```
* the shorthand syntax

```
    <button @click="methodName"></button>
```
